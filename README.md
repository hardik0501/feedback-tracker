## ðŸš€ Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/hardik0501/feedback-tracker.git
   cd feedback-tracker
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Set up environment**
   - Copy `.env.example` to `.env` and update variables.

4. **Run locally**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

5. **Visit:** [http://localhost:3000](http://localhost:3000)
